## üéâ Getting Started Right Now

### In 5 Minutes

```bash
# 1. Install
npm install -g ton-mcp

# 2. Configure Cursor (or your AI tool)
# Add to Cursor settings:
{
  "mcpServers": {
    "ton-mcp": {
      "command": "ton-mcp"
    }
  }
}

# 3. Start using!
# In Cursor, try: "Search TON docs for smart contracts"
```
